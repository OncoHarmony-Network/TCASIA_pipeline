# Output Format Reference

Complete description of output files generated by the TCASIA Alternative Splicing Analysis Pipeline.

## Directory Structure

```
workdir/
├── aligned/                        # Alignment outputs
│   ├── {sample}.Aligned.sortedByCoord.out.bam
│   ├── {sample}.Aligned.sortedByCoord.out.bam.bai
│   └── {sample}.Log.final.out
├── logs/                           # Log files
│   ├── STAR/
│   ├── rmats/
│   ├── MAJIQ/
│   ├── SUPPA2/
│   └── spladder/
├── rmats/                          # rMATS outputs
│   ├── SE.MATS.JC.txt
│   ├── A5SS.MATS.JC.txt
│   ├── A3SS.MATS.JC.txt
│   ├── MXE.MATS.JC.txt
│   ├── RI.MATS.JC.txt
│   └── summary.txt
├── MAJIQ/                          # MAJIQ outputs
│   ├── build/
│   │   ├── {sample}.majiq
│   │   └── splicegraph.sql
│   └── psi/
│       ├── {sample}.psi.tsv
│       └── {sample}.voila
├── SUPPA2/                         # SUPPA2 outputs
│   ├── salmon/
│   │   └── {sample}/
│   │       └── quant.sf
│   ├── {cohort}_SE.psi
│   ├── {cohort}_A5.psi
│   ├── {cohort}_A3.psi
│   ├── {cohort}_MX.psi
│   ├── {cohort}_RI.psi
│   └── {cohort}_AF.psi
└── spladder/                       # SplAdder outputs
    └── merge_graphs_{cohort}/
        ├── exon_skip_C3.confirmed.txt
        ├── intron_retention_C3.confirmed.txt
        ├── alt_3prime_C3.confirmed.txt
        ├── alt_5prime_C3.confirmed.txt
        └── mutex_exons_C3.confirmed.txt
```

## Alignment Outputs

### STAR BAM Files

**File**: `{sample}.Aligned.sortedByCoord.out.bam`

- **Format**: BAM (Binary Alignment/Map)
- **Sorted**: By genomic coordinate
- **Indexed**: `.bai` file included
- **Content**: Aligned RNA-seq reads

**Viewing**:
```bash
samtools view {sample}.Aligned.sortedByCoord.out.bam | head
samtools flagstat {sample}.Aligned.sortedByCoord.out.bam
```

### STAR Log Files

**File**: `{sample}.Log.final.out`

**Key metrics**:
```
Number of input reads
Uniquely mapped reads number
Uniquely mapped reads %
Number of splices: Total
Number of splices: Annotated (sjdb)
Number of splices: GT/AG
Number of splices: GC/AG
Number of splices: AT/AC
Number of splices: Non-canonical
```

**Example**:
```
                                 Number of input reads |	150000000
                     Uniquely mapped reads number |	120000000
                          Uniquely mapped reads % |	80.00%
                        Number of splices: Total |	50000000
           Number of splices: Annotated (sjdb) |	45000000
```

## rMATS Outputs

### Event Files

**Files**: `SE.MATS.JC.txt`, `A5SS.MATS.JC.txt`, `A3SS.MATS.JC.txt`, `MXE.MATS.JC.txt`, `RI.MATS.JC.txt`

**Format**: Tab-separated values (TSV)

**Event types**:
- **SE**: Skipped Exon
- **A5SS**: Alternative 5' Splice Site
- **A3SS**: Alternative 3' Splice Site
- **MXE**: Mutually Exclusive Exons
- **RI**: Retained Intron

### Column Descriptions

#### Common Columns (all event types)

| Column | Description | Example |
|--------|-------------|---------|
| `ID` | Event identifier | `1` |
| `GeneID` | Gene symbol | `ENSG00000123456.7` |
| `geneSymbol` | Gene name | `TP53` |
| `chr` | Chromosome | `chr17` |
| `strand` | Strand | `+` or `-` |

#### SE (Skipped Exon) Specific Columns

| Column | Description | Example |
|--------|-------------|---------|
| `exonStart_0base` | Exon start (0-based) | `7571720` |
| `exonEnd` | Exon end (1-based) | `7571810` |
| `upstreamES` | Upstream exon start | `7571000` |
| `upstreamEE` | Upstream exon end | `7571100` |
| `downstreamES` | Downstream exon start | `7572000` |
| `downstreamEE` | Downstream exon end | `7572100` |
| `IJC_SAMPLE_1` | Inclusion junction counts (group 1) | `10,15,20` |
| `SJC_SAMPLE_1` | Skipping junction counts (group 1) | `5,8,10` |
| `IJC_SAMPLE_2` | Inclusion junction counts (group 2) | `2,3,4` |
| `SJC_SAMPLE_2` | Skipping junction counts (group 2) | `20,25,30` |
| `IncFormLen` | Inclusion form length | `150` |
| `SkipFormLen` | Skipping form length | `60` |
| `PValue` | P-value | `0.001` |
| `FDR` | False discovery rate | `0.05` |
| `IncLevel1` | Inclusion level (group 1) | `0.67,0.65,0.67` |
| `IncLevel2` | Inclusion level (group 2) | `0.09,0.11,0.12` |
| `IncLevelDifference` | ΔPSI (group1 - group2) | `0.56` |

**PSI Calculation**:
```
PSI = IJC / (IJC + SJC)
```

**Example row**:
```
ID	GeneID	geneSymbol	chr	strand	exonStart_0base	exonEnd	upstreamES	upstreamEE	downstreamES	downstreamEE	IJC_SAMPLE_1	SJC_SAMPLE_1	IJC_SAMPLE_2	SJC_SAMPLE_2	IncFormLen	SkipFormLen	PValue	FDR	IncLevel1	IncLevel2	IncLevelDifference
1	ENSG00000141510.16	TP53	chr17	-	7571720	7571810	7571000	7571100	7572000	7572100	10,15,20	5,8,10	2,3,4	20,25,30	150	60	0.001	0.05	0.67,0.65,0.67	0.09,0.11,0.12	0.56
```

## MAJIQ Outputs

### Build Files

**File**: `{sample}.majiq`

- **Format**: Binary (MAJIQ-specific)
- **Content**: Splice graph and junction quantification
- **Usage**: Input for `majiq psi` and `majiq deltapsi`

**File**: `splicegraph.sql`

- **Format**: SQLite database
- **Content**: Splice graph structure
- **Viewing**: `sqlite3 splicegraph.sql "SELECT * FROM genes LIMIT 10;"`

### PSI Files

**File**: `{sample}.psi.tsv`

**Format**: Tab-separated values

**Columns**:

| Column | Description | Example |
|--------|-------------|---------|
| `gene_id` | Gene identifier | `ENSG00000141510` |
| `gene_name` | Gene symbol | `TP53` |
| `lsv_id` | Local Splice Variation ID | `chr17:7571720-7571810:+` |
| `junctions` | Junction coordinates | `7571000-7571720,7571810-7572000` |
| `psi` | PSI value | `0.65` |
| `confidence` | Confidence interval | `0.60-0.70` |
| `reads` | Supporting reads | `45` |

**Example**:
```
gene_id	gene_name	lsv_id	junctions	psi	confidence	reads
ENSG00000141510	TP53	chr17:7571720-7571810:+	7571000-7571720,7571810-7572000	0.65	0.60-0.70	45
```

### Voila Files

**File**: `{sample}.voila`

- **Format**: Binary (MAJIQ-specific)
- **Content**: Visualization data
- **Usage**: Input for MAJIQ Voila web interface
- **Viewing**: `voila view {sample}.voila`

## SUPPA2 Outputs

### Salmon Quantification

**File**: `salmon/{sample}/quant.sf`

**Format**: Tab-separated values

**Columns**:

| Column | Description | Example |
|--------|-------------|---------|
| `Name` | Transcript ID | `ENST00000269305.8` |
| `Length` | Transcript length | `2591` |
| `EffectiveLength` | Effective length | `2441` |
| `TPM` | Transcripts Per Million | `123.45` |
| `NumReads` | Estimated read count | `1500.5` |

### PSI Files

**Files**: `{cohort}_SE.psi`, `{cohort}_A5.psi`, `{cohort}_A3.psi`, `{cohort}_MX.psi`, `{cohort}_RI.psi`, `{cohort}_AF.psi`

**Format**: Tab-separated values

**Event types**:
- **SE**: Skipped Exon
- **A5**: Alternative 5' Splice Site
- **A3**: Alternative 3' Splice Site
- **MX**: Mutually Exclusive Exons
- **RI**: Retained Intron
- **AF**: Alternative First Exon
- **AL**: Alternative Last Exon

**Columns**:
- First column: Event ID
- Subsequent columns: PSI values for each sample

**Event ID format**:
```
{gene}:{event_type}:{coordinates}
```

**Example**:
```
event_id	sample1	sample2	sample3
ENSG00000141510;SE:chr17:7571720-7571810:+	0.65	0.70	0.60
ENSG00000141510;A5:chr17:7571720-7571810:+	0.45	0.50	0.40
```

## SplAdder Outputs

### Event Files

**Files**: `exon_skip_C3.confirmed.txt`, `intron_retention_C3.confirmed.txt`, etc.

**Format**: Tab-separated values

**Event types**:
- **exon_skip**: Exon skipping
- **intron_retention**: Intron retention
- **alt_3prime**: Alternative 3' splice site
- **alt_5prime**: Alternative 5' splice site
- **mutex_exons**: Mutually exclusive exons

### Column Descriptions

#### Exon Skip Events

| Column | Description | Example |
|--------|-------------|---------|
| `event_id` | Event identifier | `chr17_7571720_7571810_+` |
| `gene_name` | Gene symbol | `TP53` |
| `chr` | Chromosome | `chr17` |
| `strand` | Strand | `+` |
| `exon_pre_start` | Upstream exon start | `7571000` |
| `exon_pre_end` | Upstream exon end | `7571100` |
| `exon_start` | Skipped exon start | `7571720` |
| `exon_end` | Skipped exon end | `7571810` |
| `exon_aft_start` | Downstream exon start | `7572000` |
| `exon_aft_end` | Downstream exon end | `7572100` |
| `psi_sample1` | PSI for sample 1 | `0.65` |
| `psi_sample2` | PSI for sample 2 | `0.70` |
| `...` | Additional samples | `...` |

**Example**:
```
event_id	gene_name	chr	strand	exon_pre_start	exon_pre_end	exon_start	exon_end	exon_aft_start	exon_aft_end	psi_sample1	psi_sample2
chr17_7571720_7571810_+	TP53	chr17	+	7571000	7571100	7571720	7571810	7572000	7572100	0.65	0.70
```

## Data Integration

### Unified Event Format

For downstream analysis, events are typically unified:

**Unified columns**:
- `event_id`: Standardized event identifier
- `gene_id`: Ensembl gene ID
- `gene_name`: Gene symbol
- `chr`: Chromosome
- `strand`: Strand
- `event_type`: SE, A5SS, A3SS, MXE, RI
- `coordinates`: Genomic coordinates
- `psi_sample1`, `psi_sample2`, ...: PSI values per sample
- `algorithm`: Detection algorithm (rMATS, MAJIQ, SUPPA2, SplAdder)

**Example unified format**:
```
event_id	gene_id	gene_name	chr	strand	event_type	coordinates	psi_sample1	psi_sample2	algorithm
chr17:7571720-7571810:+	ENSG00000141510	TP53	chr17	+	SE	7571720-7571810	0.65	0.70	rMATS
```

## File Formats

### BAM (Binary Alignment/Map)

- **Specification**: SAM/BAM format specification v1.6
- **Tools**: samtools, IGV, UCSC Genome Browser
- **Compression**: BGZF (block gzip)

### TSV (Tab-Separated Values)

- **Delimiter**: Tab (`\t`)
- **Header**: First line (column names)
- **Missing values**: `NA` or `-`
- **Tools**: Excel, R, Python pandas

### MAJIQ Binary Formats

- **`.majiq`**: Binary splice graph
- **`.voila`**: Binary visualization data
- **Tools**: MAJIQ command-line tools, Voila web interface

## Downstream Analysis

### Loading Data in R

```r
# rMATS
rmats_se <- read.table("rmats/SE.MATS.JC.txt", header=TRUE, sep="\t")

# SUPPA2
suppa2_se <- read.table("SUPPA2/cohort_SE.psi", header=TRUE, sep="\t", row.names=1)

# SplAdder
spladder_se <- read.table("spladder/merge_graphs_cohort/exon_skip_C3.confirmed.txt",
                          header=TRUE, sep="\t")
```

### Loading Data in Python

```python
import pandas as pd

# rMATS
rmats_se = pd.read_csv("rmats/SE.MATS.JC.txt", sep="\t")

# SUPPA2
suppa2_se = pd.read_csv("SUPPA2/cohort_SE.psi", sep="\t", index_col=0)

# SplAdder
spladder_se = pd.read_csv("spladder/merge_graphs_cohort/exon_skip_C3.confirmed.txt", sep="\t")
```

## Troubleshooting

### Empty Output Files

**Issue**: Output file exists but has no events
```bash
# Check file size
ls -lh workdir/rmats/SE.MATS.JC.txt

# Count lines (should be > 1)
wc -l workdir/rmats/SE.MATS.JC.txt
```

**Possible causes**:
- Low sequencing depth
- Stringent filtering parameters
- Incorrect strandness setting
- Reference annotation mismatch

### Corrupted Files

**Issue**: Cannot read output file
```bash
# Check file integrity
file workdir/rmats/SE.MATS.JC.txt

# Verify format
head workdir/rmats/SE.MATS.JC.txt
```

**Solution**: Re-run the specific algorithm
